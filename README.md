# Como rodar

1. Faça o clone do repositorio

2. Navegue até a pasta do projeto

3. Verifique ou baixe as bibliotecas
  - PySimpleGUI

4. Rode o comando

```
python3 app.py
```

ou 

```
python app.py
```
